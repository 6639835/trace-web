# UDP flows

UDP flows show connectionless traffic that does not appear in the HTTP request list.

## View UDP flows

1. Open **Settings â†’ UDP Flows**.
2. Use search or filters to narrow by IP, port, or protocol.
3. Tap a flow to see recent datagrams and stats.

## Export UDP flows

- Use **Export CSV** or **Export JSON** from the UDP flows screen.

## Notes

- UDP traffic is not visible in proxy-only mode.
- QUIC and other UDP protocols appear here when captured.
