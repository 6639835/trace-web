# Utilities

Trace ships with on-device utilities to help you transform data while debugging.

## Available tools

- Base64 encode/decode
- URL encode/decode
- JSON formatter
- Hash generator
- Timestamp converter

## Tips

- Use the JSON formatter to validate request bodies.
- Hash tools help verify signatures or checksums.
- Timestamp conversion helps correlate logs and requests.
