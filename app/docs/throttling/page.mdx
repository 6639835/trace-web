# Network throttling

Network throttling simulates slower or unreliable connections to help you test real-world conditions.

## Enable throttling

1. Open **Settings â†’ Network Throttling**.
2. Toggle **Enable Network Throttling**.
3. Choose a preset or configure a custom profile.

## Presets and custom profiles

- Presets include Wi-Fi, 5G, 4G, 3G, 2G, satellite, and offline.
- Custom profiles let you set download, upload, latency, and packet loss.

## Notes

- Throttling affects traffic while capture is active.
- Disable throttling when you do not need it to conserve resources.
