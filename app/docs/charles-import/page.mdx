# Charles Proxy import

Trace exports standard HAR files that can be opened in Charles Proxy.

## Export from Trace

1. In Trace, open **Network**.
2. Use the menu to **Export HAR**.
3. Save the file using the iOS Share Sheet.

## Import into Charles

1. Open Charles on your desktop.
2. Choose **File â†’ Import Session**.
3. Select the HAR file exported from Trace.

## Notes and limitations

- HAR import is a one-time snapshot, not a live proxy bridge.
- Some binary bodies may be truncated if you exported with size limits.

<Tip>
  HAR captures requests, responses, headers, and timing, making it the most compatible format for desktop tooling.
</Tip>
