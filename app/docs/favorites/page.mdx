# Favorites

Favorites let you mark important requests for quick access.

## Add favorites

- Tap the star icon on a request in the Network list.
- Use the request context menu to toggle favorites.

## View favorites

1. In **Network**, open the toolbar menu.
2. Select **Favorites**.
3. Tap a favorite to open the request.

## Tips

- Favorites are great for recurring requests in a test flow.
- Pair favorites with sessions to keep test runs organized.
