# WebSocket inspector

Trace captures WebSocket connections alongside HTTP traffic.

## View WebSocket events

- WebSocket connections appear in the Network list.
- Open a WebSocket entry to view frames, metadata, and timing.

## Tips

- Use filters to isolate WebSocket domains.
- Export requests for offline analysis when needed.
