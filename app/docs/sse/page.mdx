# SSE inspector

Trace captures Server-Sent Events (SSE) streams alongside HTTP traffic.

## View SSE events

- SSE streams appear in the Network list.
- Open a stream to review events, payloads, and timing.

## Tips

- Use filters to isolate SSE endpoints.
- Save a session before clearing long-running streams.
